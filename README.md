Luminarch (루미나크)

“어둠 속, 빛만이 길을 말해준다.”
Luminarch — The Door of Light

프로젝트 개요

프로젝트명: Luminarch (루미나크) — 빛(Lumen) + Arch(출구/문)
장르: 1인칭 퍼즐 어드벤처 / 탈출형 스릴러
플랫폼: PC
개발 기간: 2025.09 ~ 2025.11 (약 2개월)
개발 인원/역할: 1인 개발 (레벨 디자인, 환경 아트, 퍼즐 시스템 C++ 구현)
툴/엔진: Unreal Engine 5, SketchUp, Photoshop, Illustrator

핵심 컨셉

한 줄 소개: 끝없는 어둠 속, 희미한 빛을 따라 탈출을 시도하라.

기획 의도:

백룸처럼 익명적이고 폐쇄적인 공간에서 오는 긴장감 전달

“빛 = 단서”라는 메커니즘으로 탐험과 퍼즐 해결을 연결

공포·심리적 압박감을 통해 공간 자체가 퍼즐이 되는 경험 제공

타겟 유저층: 퍼즐 / 스릴러 / 공포 장르를 즐기는 게이머 (15세 이상)
차별성: 어둠 속에서 빛을 찾아 해석하는 심리적 퍼즐 플레이

게임 플레이 & 조작 방식

기본 조작:
W, A, S, D - 캐릭터 이동
마우스 이동 - 시야 / 카메라 회전
E - 상호작용 (문, 스위치, 손전등 등)
Tab - 수집한 단서 / 암호 확인
F - 손전등 ON/OFF (습득 후 활성화)
Esc - 일시정지 및 메뉴 호출

플레이 흐름:
깨어남 → 폐쇄된 공간 탐험 → 손전등 습득 → 어둠 속 단서 탐색 → 퍼즐 해결 → 출구(빛의 아치) 탈출

손전등 시스템:

초반에는 손전등 사용 불가. 특정 구간에서 손전등 오브젝트를 습득해야 활성화됨.

이후 F키로 ON/OFF 가능하며 배터리 제한 시스템 적용 (HUD로 표시).

위젯 / UI 시스템:

Widget Screen 기반 UI (단서 확인, 배터리 잔량, 인터랙션 안내).

상호작용 시 화면 중앙에 메시지 표시.

빛 단서 퍼즐:

손전등을 비춰야만 확인 가능한 숨겨진 암호, 벽 문양, 바닥 표시 존재.

반사 각도, 그림자, 밝기 차이를 이용한 시각적 퍼즐 해석 필요.

환경 퍼즐:

복도 내 일부 구간은 어둠과 조명 패턴이 단서 역할을 함.

잘못된 방향 선택 시 폐쇄, 붕괴, 루프 공간으로 진입.

최종 구간:

모든 퍼즐을 해석해 문(Arch)을 열면 강렬한 백색광 등장.

밝은 공간으로 들어가면 게임 종료(탈출).

콘텐츠 구성

캐릭터: 이름 없는 실험 대상자 (1인칭 시점)

퍼즐 오브젝트:

조명 스위치, 거울 반사 장치, 그림자 암호, 잠긴 문

손전등(획득 후 사용 가능), 단서 노트 UI

맵 구조:

무한히 이어지는 폐쇄 복도, 어두운 구간, 출구 ‘빛의 아치’

목표:

빛을 활용해 숨겨진 단서를 찾아내고, 어둠의 구조를 해석해 탈출

기술적 구현 (Unreal C++)

상호작용 시스템

인터페이스 기반 설계 (IInteractable)

문, 스위치, 손전등, 단서 오브젝트에 공통 적용

손전등 시스템

C++ 기반 라이트 컴포넌트 활성/비활성 제어

배터리 소모 로직 및 UI 연동

F 입력 시 토글 처리

빛 단서 퍼즐 로직

특정 각도에서만 보이는 머티리얼 또는 Decal 사용

손전등 광원과 충돌 시 Emission 활성화

무한 복도 구조

모듈형 복도 블록 풀링

플레이어 이동 방향 기준으로 다음 구간 자동 생성

UI & 위젯 시스템

UMG로 배터리, 단서, 인터랙션 텍스트 HUD 구현

이벤트 기반 업데이트 (OnBatteryChanged, OnInteract 등)

아트 & 비주얼

그래픽 스타일: 현실적 폐허 + 어둠 대비 강한 빛 연출

조명 연출:

희미한 천장 조명, 깜빡이는 네온, 청록빛 LED

출구(Arch)는 강렬한 순백광으로 대비 연출

UI 디자인:

배터리 표시, 단서 목록, 인터랙션 메시지 표시 위젯
